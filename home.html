The two sports within wakeboarding

We can classify wakeboarding into two major types, done using a boat, and done using several towers, called "cable".
On this site I will cover the Cable part as it is the sport that I do.

Essential Gear and Learning the Basics

To start wakeboarding you need to have the essential gear, this includes a life jacket, a helmet if you are on cable, tow rope if on boat, and a wakeboard. You can learn more in the "Get Start" page!

Tricks Tutorial

Before hitting the water, spend time learning the basics of wakeboarding, including proper stance, balance, and how to handle the wakeboard in the water.
